# Child Learning Example / 儿童学习示例

## Scenario / 场景

A parent or teacher wants to help a child understand a new concept without overwhelming them.

家长或老师希望帮助儿童理解一个新概念，但不希望一上来就讲得太抽象或太复杂。

## User prompt / 用户提示词

```text
Use $mastery-coach to teach my 8-year-old what AI is. Use simple analogies, short questions, and gentle correction.
```

```text
使用 $mastery-coach 帮我向 8 岁学习者讲解什么是 AI。用简单类比、短问题和温和纠错。
```

## Expected skill behavior / 预期行为

- Use concrete analogies, such as helpers, pattern guessing, or learning from examples.
- Avoid heavy terminology.
- Teach one tiny idea at a time.
- Ask the child to explain it back.
- Use one short practice question.
- Correct gently and positively.

- 使用具体类比，例如“小助手”“猜规律”“看很多例子后变聪明”；
- 避免复杂术语；
- 一次只讲一个小概念；
- 让学习者用自己的话复述；
- 每次只给一个短问题；
- 纠错时温和、正向。

## Example flow / 示例流程

1. Explain AI as a helper that learns from many examples.
2. Ask: “If the helper sees many pictures of cats, what might it learn to notice?”
3. Let the child answer.
4. Correct or reinforce the answer.
5. Add one new idea: AI can make mistakes when it has not seen enough examples.
6. Ask the child to explain AI in one sentence.

1. 把 AI 解释成“看很多例子后会帮忙猜答案的小助手”；
2. 问：“如果小助手看了很多猫的图片，它可能会学会注意什么？”；
3. 等学习者回答；
4. 根据回答补充或纠正；
5. 增加一个新概念：如果例子不够，AI 也会猜错；
6. 让学习者用一句话复述什么是 AI。
