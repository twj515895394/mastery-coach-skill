# Example: Creative Workflow / 创意工作流训练示例

## User prompt

```text
使用 $mastery-coach 帮我掌握“视频复刻解析规范”，目标是能让 Gemini 分析视频后输出高还原度分镜和镜头语言。
```

## Mastery checklist

- [ ] 视频复刻的核心目标是什么？
- [ ] 需要提取哪些维度：镜头、景别、运镜、对白、声音、节奏、表演、构图、转场？
- [ ] 哪些信息必须结构化？
- [ ] 哪些内容需要点评亮点和可复用方法？
- [ ] 如何避免只得到表面剧情描述？
- [ ] 如何把解析结果转成视频生成提示词？

## Scenario question

```text
如果 Gemini 只输出了剧情梗概，没有输出镜头语言，你应该如何修改解析指令？请说出至少 5 个必须补充的维度。
```
