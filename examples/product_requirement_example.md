# Example: Product Requirement / 产品需求训练示例

## User prompt

```text
Use $mastery-coach to review this AI route recommendation requirement for logistics dispatch.
```

## Mastery checklist

- [ ] Who is the user: dispatcher, driver, operations, or customer service?
- [ ] What decision does the product support?
- [ ] What data is required: distance, tolls, traffic, weather, fuel price, capacity?
- [ ] What are hard constraints vs scoring factors?
- [ ] What does the recommendation output need to explain?
- [ ] What fallback is needed when real-time data is missing?
- [ ] What metrics define success?

## Questions

```text
If the fastest route has high weather risk and the safer route is 20 minutes slower, how should the system explain its recommendation?
```

```text
Which fields must be deterministic and which can be LLM-generated narrative?
```
