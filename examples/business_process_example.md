# Example: Business Process / 业务流程训练示例

## User prompt

```text
使用 $mastery-coach 帮我学习国际件从客户下单到南非派送的完整流程。
```

## Mastery checklist

- [ ] 客户下单后生成哪些核心单据？
- [ ] 国内仓入库做什么？
- [ ] 国内出口和国内报关的边界是什么？
- [ ] 空运/海运节点如何影响时效？
- [ ] 南非清关涉及哪些状态？
- [ ] 机场仓入库、出库和承运商派送如何衔接？
- [ ] 付税失败、清关异常、派送失败如何处理？
- [ ] 哪些节点适合 AI Agent 辅助？

## Scenario question

```text
如果包裹已经到达 JNB，但付税状态异常，机场仓是否应该出库？为什么？涉及哪些系统状态联动？
```
