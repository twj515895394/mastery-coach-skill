# Mastery Checklist / 掌握清单

Topic / 主题: `{{topic}}`
Mode / 模式: `{{mode}}`
Target mastery level / 目标掌握等级: `{{level}}`

## 1. Problem / 问题
- [ ] What is the problem?
- [ ] Why does it matter?
- [ ] What happens if it is misunderstood?

## 2. Context / 背景
- [ ] What background knowledge is required?
- [ ] What assumptions are involved?
- [ ] Who or what is affected?

## 3. Mechanism / 机制
- [ ] How does it work?
- [ ] What are the key components?
- [ ] What is the end-to-end flow?

## 4. Decisions / 决策
- [ ] Why this solution or interpretation?
- [ ] What alternatives exist?
- [ ] What tradeoffs were made?

## 5. Edge Cases / 边界情况
- [ ] What can go wrong?
- [ ] What are the failure modes?
- [ ] How should exceptions be handled?

## 6. Verification / 验证
- [ ] What questions prove understanding?
- [ ] What examples or scenarios should the user handle?
- [ ] What evidence shows the explanation is correct?

## 7. Transfer / 迁移
- [ ] Where else can this idea apply?
- [ ] What similar problems can now be solved?
- [ ] Can the user teach this to someone else?
