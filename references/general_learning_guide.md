# General Learning Guide / 通用学习指南

This guide helps Mastery Coach adapt to learners of different ages, backgrounds, and goals.

本指南用于帮助 Mastery Coach 适配不同年龄、不同基础、不同学习目标的人群。

## Learner profiles / 学习者画像

| Learner | Teaching style | Mastery check |
|---|---|---|
| Young child | Story, concrete object, analogy, short turns | Ask for a simple example or picture-like explanation |
| Teenager | School task, project, exam, real-life decision | Ask them to solve a similar problem or explain a step |
| Adult beginner | Usefulness first, clear structure, respectful tone | Ask them to apply the idea to work, life, or a realistic case |
| Professional learner | Compact, precise, edge cases, transfer | Ask tradeoff, critique, and transfer questions |
| Parent | Facilitation prompts and gentle correction language | Ask how they would guide the learner next |
| Teacher | Lesson stages, classroom questions, review drills | Ask for the lesson objective and evidence of understanding |

| 学习者 | 教学风格 | 掌握度检查 |
|---|---|---|
| 低龄儿童 | 故事、具体物品、类比、短回合 | 让学习者举一个简单例子，或用画面感语言复述 |
| 青少年 | 学业任务、项目、考试、现实选择 | 让学习者解决一个类似问题，或解释关键步骤 |
| 成人初学者 | 先讲用途，结构清晰，语气尊重 | 让学习者把知识应用到工作、生活或真实案例 |
| 专业学习者 | 精准压缩，关注边界、迁移和取舍 | 提问取舍、批判和迁移问题 |
| 家长 | 给引导话术和温和纠错表达 | 询问下一步会如何引导学习者 |
| 老师 | 课程阶段、课堂提问、复习训练 | 明确教学目标和理解证据 |

## General learning loop / 通用学习闭环

1. Identify the learner and learning target.
2. Clarify what counts as mastery.
3. Explain one small chunk.
4. Ask for a teach-back.
5. Give one realistic or age-appropriate practice question.
6. Diagnose the answer.
7. Correct one gap at a time.
8. Summarize and move to the next chunk.

1. 识别学习者和学习目标。
2. 明确什么才算掌握。
3. 只讲一个小知识块。
4. 让学习者复述。
5. 给一个符合年龄和场景的练习题。
6. 诊断回答。
7. 一次只修正一个缺口。
8. 总结后进入下一小节。
