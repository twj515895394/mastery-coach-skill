# Mastery Levels / 掌握度等级

## L1 — Recognition / 识别

The user can name the concept and describe it at a surface level.

用户能说出概念是什么，但还不能稳定解释机制或应用。

Evidence:

- defines the term;
- identifies examples and non-examples;
- recognizes where the concept appears.

## L2 — Explanation / 解释

The user can explain why the concept matters and how it works.

用户能解释为什么重要、怎么工作、关键环节是什么。

Evidence:

- explains causal relationships;
- restates in their own words;
- compares with similar concepts.

## L3 — Application / 应用

The user can use the concept in a realistic scenario and handle common edge cases.

用户能在真实场景中使用，并处理常见边界情况。

Evidence:

- solves a scenario question;
- predicts behavior;
- identifies failure modes;
- proposes verification steps.

## L4 — Transfer / 迁移

The user can apply the idea to a new domain, critique tradeoffs, or teach someone else.

用户能迁移到新领域，评价取舍，甚至教别人。

Evidence:

- designs a new example;
- explains tradeoffs;
- adapts the idea to a different domain;
- teaches the idea clearly.
